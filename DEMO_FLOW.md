# 🎮 Cedra Quest - Demo Flow Guide

## 📱 **App Overview**
Cedra Quest là một Telegram Mini App kết hợp game, pet care, và blockchain rewards. Người dùng có thể chơi game, nuôi pet, hoàn thành quest để kiếm điểm và rank up.

---

## 🚀 **Demo Flow - Step by Step**

### **1. 🏠 Home Screen - First Impression**
**Mục đích**: Giới thiệu app và hiển thị thông tin cơ bản

**Nội dung hiển thị**:
- **User Info**: Avatar, username, level, XP
- **Balance**: Total points (coins), gem balance
- **Quick Stats**: Current rank, energy level
- **Navigation**: Bottom tabs (Home, Quest, Pet, Wallet, Game)

**Demo Actions**:
- Chào mừng user mới
- Hiển thị balance: 0 points ban đầu
- Giải thích các tab chính

---

### **2. 🥚 Pet Screen - Pet Hatching Journey**
**Mục đích**: Hướng dẫn user hoàn thành pet quest để hatch pet

**Flow Demo**:

#### **Step 2.1: Egg State**
- Hiển thị trứng pet chưa nở
- Progress bar: 0% → 100%
- Button "Hatch" để xem nhiệm vụ

#### **Step 2.2: Pet Quest Modal**
**4 nhiệm vụ cần hoàn thành**:
1. ✅ **Follow Twitter** → +25% progress
2. ✅ **Join Telegram** → +25% progress  
3. ✅ **Invite Friend** → +25% progress
4. ✅ **Enter Birth Year** → +25% progress

#### **Step 2.3: Pet Hatched**
- Animation trứng vỡ → pet xuất hiện
- Pet dựa trên năm sinh (12 con giáp)
- Bắt đầu generate coins mỗi phút

#### **Step 2.4: Pet Care System**
- **Mission Button**: Feed pet (20 points → +20 XP)
- **Care Button**: Mua care items (tăng XP nhanh)
- **Friends Button**: Invite system
- **Claim Coins**: Thu hoạch coins từ pet mining

---

### **3. 🎯 Quest Screen - Earning Points**
**Mục đích**: Hướng dẫn cách kiếm points qua quest system

**Demo Flow**:

#### **Step 3.1: Daily Quests**
- **Login Daily**: 100 points
- **Play 3 Games**: 150 points  
- **Feed Pet 5 Times**: 200 points
- **Invite 1 Friend**: 300 points

#### **Step 3.2: Task Quests**
- **Follow Social Media**: 500 points
- **Join Community**: 300 points
- **Share App**: 200 points
- **Complete Profile**: 100 points

#### **Step 3.3: Achievement Quests**
- **Reach Level 5**: 1000 points
- **Hatch Pet**: 500 points
- **Play 50 Games**: 2000 points
- **Earn 10K Points**: 5000 points

**Demo Actions**:
- Claim một vài quest để có points
- Hiển thị animation claim reward
- Points tăng trong balance

---

### **4. 🎮 Game Screen - Pocket Fly Game**
**Mục đích**: Demo gameplay và earning system

**Demo Flow**:

#### **Step 4.1: Game Menu**
- Hiển thị energy: 10/10
- Best score, total games played
- Button "Play" để bắt đầu

#### **Step 4.2: Gameplay**
- **Pocket Fly Game**: Flappy bird style
- Điều khiển: Tap để bay
- Mục tiêu: Tránh pipes, ghi điểm cao

#### **Step 4.3: Game Over**
- Hiển thị score: ví dụ 15 điểm
- Points earned: 15 điểm (1:1 ratio)
- Energy consumed: -1
- Button "Play Again" hoặc "Back to Menu"

**Demo Actions**:
- Chơi 2-3 game để demo
- Hiển thị points tăng sau mỗi game
- Energy giảm dần

---

### **5. 🏆 Ranking System - Progress Tracking**
**Mục đích**: Demo hệ thống rank và rewards

**Demo Flow**:

#### **Step 5.1: Current Rank**
- **RANK1**: 0 points (starting rank)
- Progress bar: current points / next rank threshold
- Next rank: RANK2 at 10,000 points

#### **Step 5.2: Rank Up Demo**
- Accumulate points qua quest + game + pet
- Khi đạt 10,000 points → RANK2
- **Rank Up Animation**: 🎉 + 1,000 coins reward
- New threshold: RANK3 at 25,000 points

#### **Step 5.3: Rank Benefits**
- Higher ranks = more pet coin generation
- Exclusive quests unlock
- Special rewards

---

### **6. 💰 Wallet Screen - Blockchain Integration**
**Mục đích**: Demo wallet connection và rewards

**Demo Flow**:

#### **Step 6.1: Wallet Connection**
- Connect Petra/Martian wallet
- Display wallet address
- Show CEDRA token balance

#### **Step 6.2: Rewards Claim**
- Convert in-game points → CEDRA tokens
- Minimum claim: 1,000 points
- Transaction on Aptos blockchain

#### **Step 6.3: Transaction History**
- List of claims/rewards
- Blockchain transaction links
- Balance updates

## 🎯 **Complete Demo Scenario (15 minutes)**

### **Minute 1-2: Onboarding**
1. Open app → Home screen
2. Navigate to Pet → See egg
3. Click "Hatch" → Pet Quest Modal
4. Complete 4 tasks → Pet hatches

### **Minute 3-5: First Gameplay**
1. Navigate to Game → Play Pocket Fly
2. Score 10-20 points → Earn points
3. Play 2-3 games → Energy depletes
4. Return to Pet → Claim first coins

### **Minute 6-8: Quest System**
1. Navigate to Quest → See available quests
2. Claim daily login → +100 points
3. Complete 1-2 easy tasks → +500 points
4. Check balance increase

### **Minute 9-11: Pet Care**
1. Return to Pet → Feed pet (spend 20 points)
2. Pet gains XP → Level up animation
3. Buy care item → More XP gain
4. Claim accumulated coins

### **Minute 12-13: Ranking**
1. Check current rank progress
2. If enough points → Rank up animation
3. Receive rank reward coins

### **Minute 14-15: Wallet Demo**
1. Navigate to Wallet
2. Connect wallet (if available)
3. Show potential CEDRA claim
4. Explain blockchain rewards
