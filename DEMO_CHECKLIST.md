## 🚨 **Troubleshooting Guide**

### **Common Issues & Solutions**

**App won't load:**
- [ ] Check internet connection
- [ ] Restart Telegram app
- [ ] Clear Telegram cache
- [ ] Use backup device

**Backend API errors:**
- [ ] Check server status
- [ ] Restart backend service
- [ ] Use offline mode explanation
- [ ] Switch to backup slides

**Demo data missing:**
- [ ] Re-run demo-setup.sql
- [ ] Check database connection
- [ ] Use manual data entry
- [ ] Explain features without live demo

**Game performance issues:**
- [ ] Close other apps
- [ ] Reduce animation quality
- [ ] Use different device
- [ ] Show pre-recorded gameplay
